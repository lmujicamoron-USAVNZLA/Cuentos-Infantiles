# 📚 Buscador de Cuentos Infantiles

Proyecto interactivo para que los niños encuentren a sus personajes favoritos de cuentos clásicos.

## ✨ Características
* **Búsqueda Exacta:** Encuentra la imagen escribiendo el nombre del personaje.
* **Galería de Selección:** Haz clic en las miniaturas para ver al protagonista en grande.
* **Diseño Amigable:** Colores y formas pensadas para un público infantil.

## 🛠️ Instrucciones de Uso
1. Escribe el nombre del personaje (ej: `pinocho`) en el buscador.
2. O simplemente haz clic en uno de los círculos de la galería.
